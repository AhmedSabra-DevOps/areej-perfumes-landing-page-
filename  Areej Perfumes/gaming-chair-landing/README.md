# 🌟 أريج للعطور - Areej Perfumes (Landing Page)

صفحة هبوط (Landing Page) احترافية، سريعة ومتجاوبة بالكامل لمتجر عطور مصري فاخر، تم بناؤها باستخدام تقنيات الـ Front-End الحديثة لتوفير تجربة مستخدم سلسة وجذابة تدعم المبيعات المباشرة.

---

## 🚀 المميزات (Features)

- **تصميم فاخر وعصري (Premium Dark Theme):** واجهة جذابة باللون الأسود مدمجة مع توهجات ذهبية وعنبرية تناسب فخامة براند العطور.
- **تجاوب كامل (100% Responsive):** متوافق تماماً مع جميع الشاشات (الهواتف المحمولة، التابلت، أجهزة الكمبيوتر).
- **نظام طلب ذكي وعبر WhatsApp:** يتيح للمستخدم طلب عطر محدد بضغطة زر، لتفتح له محادثة الواتساب تلقائياً برسالة مجهزة باسم المنتج.
- **تأثيرات حركية سلسة:** إضافة تأثيرات أنيميشن مميزة وكروت زجاجية (Glassmorphism) مبهجة أثناء التصفح.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **React.js** (مع **Vite** لسرعة وأداء أفضل في التطوير والبناء).
- **Tailwind CSS** (لتنسيق الواجهات وبناء تصميم متجاوب وسريع).
- **Framer Motion** (لإضافة الأنيميشن والتأثيرات الحركية السلسة).
- **React Icons** (لاستخدام أيقونات التواصل والمميزات بجودة عالية).
